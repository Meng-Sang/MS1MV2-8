# MS1MV2-8
This dataset consists of 3724 images 8 identities. But this dataset is not given with specific image data. We give two npy files (toy.npy and toy_label.npy) of shape (3724, ). toy.npy stores the serial number of the image corresponding to it in the MS1MS2 dataset, and toy_label.npy stores the labels (0-7) corresponding to it in this dataset.
